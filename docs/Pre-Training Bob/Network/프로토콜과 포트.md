# 포트
- 0~1023: 잘 알려진 포트
- 1024~49151: 등록된 포트
- 49152~65536: 동적 포트

0번: UDP  
1번: TCP  
80번(TCP,UDP): HTTP(HyperText Transfer Protocol)  
25번(TCP): SMTP(Simple Mail Transfer Protocol)  
21번(TCP): FTP-제어포트  
20번(TCP): FTP-데이터 포트  
443번(TCP): HTTPS(HTTP+Secure Socket Layer)
